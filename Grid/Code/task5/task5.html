justify-items justify-self
justify-items is applied to the grid container and sets the default horizontal alignment for all grid items. justify-self is applied to individual grid items and allows you to override the default alignment set by justify-items.
In summary, justify-items sets the default horizontal alignment for all grid items within the grid container, while justify-self allows you to override that default alignment for individual items.
